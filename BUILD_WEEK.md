## Intended architecture

```text
Incident statement
        ↓
GPT-5.6 structured proposal
        ↓
Evidence validator + overreach guard
        ↓
Correction Before Commitment
        ↓
Approved ReceiptItem[]
        ↓
Existing deterministic receipt printer
```

GPT-5.6 proposes the paperwork. Deterministic software validates it. The user corrects and authorizes the record. The existing receipt compiler renders the final artifact.
